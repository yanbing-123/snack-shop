# 🍿 零食铺子（Snack Shop）

> 基于 HTML5 + CSS3 + Vanilla JS 的纯前端零食购物系统，无需后端，浏览器直接打开即可运行。

## 功能特性

- 🍬 **12种零食**：草莓奶糖/橘子软糖/黄油曲奇/苏打饼干/开心果/混合坚果/牛奶巧克力/黑巧克块/可乐/柠檬茶/芒果果冻/草莓布丁
- 🏷️ **6分类筛选**：全部/糖果/饼干/坚果/巧克力/饮料/果冻（即时切换，完全重绘）
- 🔍 **实时搜索**：关键词按名称匹配，搜索即时生效
- 🛒 **侧边购物车抽屉**：+/-数量/删除/清空确认/实时总价/底部固定结算栏
- 💾 **localStorage 持久化**：`snack_stock` + `snack_cart`
- 🔒 **三层防超卖**：addToCart / updateQuantity / submitOrder 全库存校验
- 📦 **结算表单**：姓名/手机(/^1[3-9]\d{9}$/)/地址验证 + 订单成功页（订单号+明细）
- 📱 **响应式**：手机2列 → 平板3列 → PC5列自适应
- 🎨 **活泼粉色主题**：粉色渐变(#FF69B4)+奶白色背景(#FFF8F0)+emoji图标
- 🔌 **window._snack** 暴露所有关键函数

## 运行方式

直接在浏览器打开 `index.html`，无需任何服务器或构建工具。

## 技术栈

- HTML5 + CSS3（Flexbox + Grid + CSS Variables）

- localStorage API
